-- -- -- -- -- -- -- -- -- -- -- -- KEYBOARD SHORTCUTS -- -- -- -- -- -- -- -- -- -- -- --

In the Part Editor

-- GENERAL --

s: Save
w: Create wiring point at cursor
z: Undo
x: Redo
SPACE: Zoom in
SHIFT + SPACE: Zoom out
RIGHT ARROW: Nudge object right
LEFT ARROW: Nudge object left
UP ARROW: Nudge object up
DOWN ARROW: Nudge object down
SHIFT + UP ARROW: Increase object scale
SHIFT + DOWN ARROW: Decrease object scale

-- WHILE HOVERING OVER A WIRING POINT --

BACKSPACE: Remove wiring point
RIGHT ARROW: Nudge wiring point right
LEFT ARROW: Nudge wiring point left
UP ARROW: Nudge wiring point up
DOWN ARROW: Nudge wiring point down

-- WHILE POP-UP WINDOW IS OPEN --

ESCAPE: Clicks on the left button (Exit pop-up)
ENTER: Clicks on the right button (Confirms action)

In the Created Image Viewer

-- GENERAL --

ESC: Return to Workspace
SPACE: Zoom in
SHIFT + SPACE: Zoom out

In the Workspace

-- GENERAL --

s: Save
z: Undo
x: Redo
c: Toggle pen mode
a: Box select
SPACE: Zoom in
SHIFT + SPACE: Zoom out
SHIFT: Hold to select multiple objects
SHIFT + UP ARROW: Increase object scale
SHIFT + DOWN ARROW: Decrease object scale

-- WHILE HOVERING OVER AN OBJECT OR WHILE OBJECT PANEL IS OPEN --

BACKSPACE: Delete object
ESCAPE: Close panel
RIGHT ARROW: Nudge object right
LEFT ARROW: Nudge object left
UP ARROW: Nudge object up
DOWN ARROW: Nudge object down
SHIFT + RIGHT ARROW: Rotate 90 degrees clockwise
SHIFT + LEFT ARROW: Rotate 90 degrees counterclockwise
SHIFT + UP ARROW: Bring to front
SHIFT + DOWN ARROW: Send to back

-- WHILE HOVERING OVER A WIRE OR WHILE WIRE PANEL IS OPEN --

BACKSPACE: Delete wire
ESCAPE: Close panel
w: Create node on wire
1: Change wire color to red
2: Change wire color to orange
3: Change wire color to yellow
4: Change wire color to green
5: Change wire color to teal
6: Chane wire color to blue
7: Change wire color to purple
8: Change wire color to pink
9: Change wire color to brown
0: Change wire color to black
SHIFT + 1: Change wire color to custom color slot 1
SHIFT + 2: Change wire color to custom color slot 2
SHIFT + 3: Change wire color to custom color slot 3
SHIFT + 4: Change wire color to custom color slot 4
SHIFT + 5: Change wire color to custom color slot 5
SHIFT + 6: Change wire color to custom color slot 6
SHIFT + 7: Change wire color to custom color slot 7
SHIFT + 8: Change wire color to custom color slot 8
SHIFT + 9: Change wire color to custom color slot 9

-- WHILE HOVERING OVER A NODE --

RIGHT ARROW: Nudge node right
LEFT ARROW: Nudge node left
UP ARROW: Nudge node up
DOWN ARROW: Nudge node down

-- WHILE HOVERING OVER A CUSTOM COLOR --

BACKSPACE: Remove custom color

-- WHILE POP-UP WINDOW IS OPEN --

ESCAPE: Clicks on the left button (Exit pop-up)
ENTER: Clicks on the right button (Confirms action)
